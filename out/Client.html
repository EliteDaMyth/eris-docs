 "jpg" The default format to provide user avatars, guild icons, and group icons in. Can be "jpg", "png", "gif", or "webp"

 128 The default size to return user avatars, guild icons, banners, splashes, and group icons. Can be any power of two between 16 and 2048. If the height and width are different, the width will be the value specified, and the height relative to that
